## 一、已修复的问题

### 1. LocaleDateException 错误
- **问题**: 打开 APP 首页报 `Locale data has not been initialized`
- **原因**: `table_calendar` 使用 `locale: 'zh_CN'` 但未初始化中文 locale 数据
- **修复**: `main.dart` 添加 `await initializeDateFormatting('zh_CN')`

### 2. 日历切换日期断言错误
- **问题**: 切换日期时出现 `Assertion failed` 闪烁
- **原因**: `table_calendar` 默认 `defaultDecoration`/`weekendDecoration` 使用 `BoxShape.circle`，与自定义的 `borderRadius` 在 `AnimatedContainer` 插值时冲突
- **修复**: `CalendarStyle` 统一使用 `borderRadius` 装饰，并对 `focusedDay` 做边界钳制

### 3. 通知服务构建错误
- **问题**: `notification_service.dart` 缺少 `uiLocalNotificationDateInterpretation` 参数
- **修复**: 添加 `uiLocalNotificationDateInterpretation: UILocalNotificationDateInterpretation.absoluteTime`

## 二、功能新增

### 1. 统计页面实时更新
- **根因**: `StatsScreen` 直接调用 `repo.getCheckinCountByDate()`，未通过 Provider watch
- **修复**: 添加 `statsDataProvider`，依赖 `calendarRecordsProvider` 触发重建
- `totalStatsProvider` 和 `achievementsProvider` 也添加了对 `calendarRecordsProvider` 的 watch

### 2. 切换日期显示对应任务
- **根因**: 首页始终用 `todayRecordsProvider`（硬编码今天）
- **修复**: 使用 `selectedDateRecordsProvider(_selectedDate)` 替代
- 添加 `isCheckedInOnDateProvider` 支持按日期查询打卡状态
- Repository 添加 `checkInForDate()` 和 `cancelCheckInForDate()` 方法

### 3. 时间分类功能
- `CheckInItem` 添加 `@HiveField(10) int frequency`（0=每天/1=每周/2=每月/3=长期/4=自定义）
- `CheckInItem` 添加 `@HiveField(11) List<int> customDays`（自定义频率的每月日期）
- Repository 添加 `getItemsForDate()` 和 `_isItemVisibleOnDate()` 按频率过滤
- Provider 添加 `itemsForDateProvider` 按日期+频率过滤任务
- AddProjectDialog 添加频率选择 UI（每天/每周/每月/长期/自定义）
- 自定义模式下显示 1-31 日期多选网格

### 4. 教程引导页
- 创建 `OnboardingScreen`（4 页滑动引导：欢迎→打卡→日历统计→成就）
- SettingsRepository/SettingsNotifier 添加 `hasCompletedOnboarding` 字段
- app.dart 条件路由：SplashScreen → OnboardingScreen（首次）→ 主界面

### 5. 应用图标
- 用 Dart `image` 包生成 1024x1024 PNG 图标（白色背景 + 蓝紫渐变圆形 + 白色对勾）
- 配置 `flutter_launcher_icons` 自动生成 Android/Web 各尺寸图标
- 生成脚本: `tool/generate_icon.dart`

### 6. 启动页
- 创建 `SplashScreen` Widget（蓝紫渐变背景 + 图标 + 应用名 + slogan）
- 弹性缩放 + 淡入动画，2 秒后自动跳转
- Android 原生启动背景改为渐变色

### 7. 右滑打卡 + 双区域列表
- 未完成任务用 `Dismissible` 包裹
- 右滑（从左到右）= 打卡，任务移到"已完成"区域
- 已完成区域显示在底部

### 8. 长按编辑 + 左滑延迟
- `CheckInCard` 添加 `onLongPress` 回调
- 长按弹出编辑对话框（修改名称/删除）
- 左滑（从右到左）= 延迟到明天（内存 Provider，重启后清除）

### 9. 深色模式
- `AppColors` 添加 `isDark` 标志 + 深色背景/文字/渐变色值（getter 动态返回）
- `AppTheme` 添加 `darkTheme`
- app.dart 设置 `darkTheme` + `themeMode`
- 设置页面新增"深色模式"开关

### 10. 提醒通知集成
- `main.dart` 初始化 `NotificationService`
- 设置页面：打开提醒后显示时间选择器（TimePicker），选择后注册系统通知
- 关闭提醒时调用 `cancelAll()`

### 11. 代码清理
- 删除未使用的 `export_service.dart`
- 全部 `.withOpacity()` 替换为 `.withValues(alpha:)`
- 移除所有使用动态 `AppColors` 的 `const` 构造器
